Python Truth Value Testing
- False and None evaluate to False
- Numeric zero values evaluate to False: 0, 0.0, 0j
- Decimal(0), Fraction(0, x) evel to False
- Empty sequences/collections: '', (), [], {} eval to False

3 basic Bool operations
- X and Y               --> true if X and Y are both true
- X or Y                --> true if X or Y are true
- not X                 --> If X is false, then true, else false
